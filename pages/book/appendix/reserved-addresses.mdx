# 부록 B: 예약된 주소

예약된 주소는 Sui에서 특정 목적을 가진 특별한 주소입니다. 환경 간에 동일하게 유지되며
특정 네이티브 작업에 사용됩니다.

- `0x1` - [표준 라이브러리](./../move-basics/standard-library) 주소 (별칭 `std`)
- `0x2` - [Sui Framework](./../programmability/sui-framework) 주소 (별칭 `sui`)
- `0x5` - `SuiSystem` object 주소
- `0x6` - 시스템 [`Clock` object](./../programmability/epoch-and-time) 주소
- `0x8` - 시스템 `Random` object 주소
- `0x403` - `DenyList` 시스템 object 주소
