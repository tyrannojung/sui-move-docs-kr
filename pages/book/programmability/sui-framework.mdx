# Sui 프레임워크

Sui 프레임워크는 [패키지 매니페스트](./../concepts/manifest)에 설정된 기본 의존성입니다. 이는
[표준 라이브러리](./../move-basics/standard-library)에 의존하며 스토리지와의 상호작용,
Sui 특화 네이티브 타입과 모듈을 포함한 Sui 특화 기능들을 제공합니다.

_편의를 위해 Sui 프레임워크의 모듈들을 여러 카테고리로 그룹화했습니다. 하지만 이들은
여전히 동일한 프레임워크의 일부입니다._

## 내보낸 주소

Sui 프레임워크는 두 개의 명명된 주소를 내보냅니다: `sui = 0x2`와 std 의존성에서 가져온 `std = 0x1`.

```toml
[addresses]
sui = "0x2"

# MoveStdlib 의존성에서 내보냄
std = "0x1"
```

## 암시적 임포트

[표준 라이브러리](./../move-basics/standard-library#implicit-imports)와 마찬가지로, Sui 프레임워크에서도
일부 모듈과 타입이 암시적으로 임포트됩니다. 다음은 명시적인 `use` 임포트 없이도
사용할 수 있는 모듈과 타입의 목록입니다:

- sui::object
- sui::object::ID
- sui::object::UID
- sui::tx_context
- sui::tx_context::TxContext
- sui::transfer
