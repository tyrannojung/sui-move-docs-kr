# 표준 라이브러리

Move 표준 라이브러리는 네이티브 타입과 연산을 위한 기능을 제공합니다. 스토리지와 상호작용하지 않지만
데이터 작업과 조작을 위한 기본 도구를 제공하는 표준 module 컬렉션입니다. 이는
[Sui Framework](./../programmability/sui-framework)의 유일한 의존성이며, 함께 가져옵니다.

어떤 module이 이를 구현하는지 파악할 수 있도록 개요를 제공하는 것도 도움이 됩니다.

## 정수 Module

Move 표준 라이브러리는 정수 타입과 관련된 함수 세트를 제공합니다. 이러한 함수들은
여러 module로 분할되어 있으며, 각각 특정 정수 타입과 연결되어 있습니다. 이 module들은
직접 가져올 필요가 없으며, 그들의 함수는 모든 정수 값에서 사용할 수 있습니다.

> 모든 module은 동일한 함수 세트를 제공합니다. 즉, `max`, `diff`,
> `divide_and_round_up`, `sqrt`, `pow`입니다.

## 내보낸 주소

표준 라이브러리는 단일 명명된 주소 - `std = 0x1`을 내보냅니다. 여기에서 `std` 별칭이 정의되어 있습니다.

```toml
[addresses]
std = "0x1"
```

## 암시적 Import

일부 module은 암시적으로 가져와지며 명시적인 `use`
import 없이 module에서 사용할 수 있습니다. 표준 라이브러리의 경우, 다음 module과 타입이 포함됩니다:

- std::vector
- std::option
- std::option::Option

## Sui Framework 없이 std 가져오기

Move 표준 라이브러리는 패키지에 직접 가져올 수 있습니다. 하지만 `std`만으로는
의미 있는 애플리케이션을 구축하기에 충분하지 않습니다. 스토리지 기능을 제공하지 않고
온체인 상태와 상호작용할 수 없기 때문입니다.

```toml
MoveStdlib = { git = "https://github.com/MystenLabs/sui.git", subdir = "crates/sui-framework/packages/move-stdlib", rev = "framework/mainnet" }
```
