Move는 주소를 나타내기 위해 [address](./../concepts/address)라는 특별한 타입을 사용합니다. 이는 블록체인의 모든 주소를 나타낼 수 있는 32바이트 값입니다. 주소는 두 가지 형식으로 작성할 수 있습니다: 0x로 시작하는 16진수 주소와 명명된 주소.

```move
// 주소 리터럴
let value: address = @0x1;

// Move.toml에 등록된 명명된 주소
let value = @std;
let other = @sui;
```

주소 리터럴은 `@` 기호로 시작하며 16진수 또는 식별자가 뒤따릅니다. 16진수는 32바이트 값으로 해석됩니다. 식별자는 [Move.toml](./../concepts/manifest) 파일에서 조회되고 컴파일러에 의해 해당 주소로 대체됩니다. Move.toml 파일에서 식별자를 찾을 수 없으면 컴파일러가 오류를 발생시킵니다.

## 변환

Sui 프레임워크는 주소 작업을 위한 헬퍼 함수 세트를 제공합니다. 주소 타입이 32바이트 값이므로 `u256` 타입으로 변환할 수 있고 그 반대도 가능합니다. 또한 `vector<u8>` 타입으로 변환하거나 그 반대로 변환할 수도 있습니다.

예제: 주소를 `u256` 타입으로 변환하고 다시 되돌리기.

```move
use sui::address;

let addr_as_u256: u256 = address::to_u256(@0x1);
let addr = address::from_u256(addr_as_u256);
```

예제: 주소를 `vector<u8>` 타입으로 변환하고 다시 되돌리기.

```move
use sui::address;

let addr_as_u8: vector<u8> = address::to_bytes(@0x1);
let addr = address::from_bytes(addr_as_u8);
```

예제: 주소를 문자열로 변환하기.

```move
use sui::address;
use std::string::String;

let addr_as_string: String = address::to_string(@0x1);
```
