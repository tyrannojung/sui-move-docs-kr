# Object Model

이 장에서는 Sui의 Object Model을 설명합니다. Object Model의 이론과 개념에 중점을 두어
Sui 스토리지 작업과 리소스 소유권에 대한 실무적인 내용을 다룰 준비를 합니다.
편의성과 쉬운 검색을 위해 이 장을 여러 섹션으로 나누어 각각 Object Model의
특정 측면을 다룹니다.

<div class="warning">

이 장은 결코 Object Model에 대한 포괄적인 가이드가 아닙니다. Object Model의
개념과 원칙에 대한 고수준 개요일 뿐입니다.

더 자세한 설명은 
[Sui 문서](https://docs.sui.io/concepts/object-model)를 참조하세요.

</div>
