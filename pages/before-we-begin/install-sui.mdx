# Sui 설치하기

Move는 컴파일 언어이므로 Move 프로그램을 작성하고 실행하려면 컴파일러를 설치해야 합니다. 컴파일러는 Sui 바이너리에 포함되어 있으며, 아래 방법 중 하나를 사용하여 설치하거나 다운로드할 수 있습니다.

## 바이너리 다운로드

[릴리스 페이지](https://github.com/MystenLabs/sui/releases)에서 최신 Sui 바이너리를 다운로드할 수 있습니다. 바이너리는 macOS, Linux 및 Windows에서 사용할 수 있습니다. 교육 목적과 개발을 위해서는 `mainnet` 버전을 사용하는 것을 권장합니다.

## Homebrew를 사용한 설치 (MacOS)

[Homebrew](https://brew.sh/) 패키지 관리자를 사용하여 Sui를 설치할 수 있습니다.

```bash
brew install sui
```

## Chocolatey를 사용한 설치 (Windows)

Windows용 [Chocolatey](https://chocolatey.org/install) 패키지 관리자를 사용하여 Sui를 설치할 수 있습니다.

```bash
choco install sui
```

## Cargo를 사용한 빌드 (MacOS, Linux)

Cargo 패키지 관리자를 사용하여 Sui를 로컬에서 설치하고 빌드할 수 있습니다 (Rust 필요)

```bash
cargo install --git https://github.com/MystenLabs/sui.git sui --branch mainnet
```

`testnet` 또는 `devnet`을 대상으로 하는 경우 여기에서 브랜치 대상을 변경하세요.

아래 명령어로 시스템에 최신 Rust 버전이 있는지 확인하세요.

```bash
rustup update stable
```

## 문제 해결

설치 과정에서 문제가 발생하면 [Sui 설치](https://docs.sui.io/guides/developer/getting-started/sui-install) 가이드를 참조하세요.
