# 원시 타입

원시 타입은 언어의 기본 구성 요소입니다.

이러한 원시 타입은 단독으로 사용되거나 더 복잡한 사용자 정의 타입을 구축하는 데 사용될 수 있습니다.
예를 들어 [`struct`](./structs)에서 사용됩니다.

- [정수](./primitive-types/integers)
- [Bool](./primitive-types/bool)
- [주소](./primitive-types/address)
- [벡터](./primitive-types/vector)

이러한 원시 타입은 다른 타입과 함께 사용됩니다

- [참조](./primitive-types/references)
- [튜플과 Unit](./primitive-types/tuples)
