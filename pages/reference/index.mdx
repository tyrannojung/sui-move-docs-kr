# Move 레퍼런스

Move에 오신 것을 환영합니다. Move는 안전한 자산 프로그래밍을 위한 차세대 언어입니다. 주요 사용 사례는
블록체인 환경으로, Move 프로그램이 상태 변경을 구성하는 데 사용됩니다. Move는
개발자가 자산을 유연하게 관리하고 전송하는 프로그램을 작성할 수 있도록 하면서도 해당 자산에 대한
공격으로부터 보안과 보호를 제공합니다. 그러나 Move는 블록체인 컨텍스트 외부의 사용 사례도
염두에 두고 개발되었습니다.

Move는 디지털 자산(예: 통화)의 명시적 표현으로 이동(move, 따라서 이름이 유래) 의미론을 가진
리소스 타입을 사용하여 [Rust](https://www.rust-lang.org/)에서 영감을 받았습니다.
