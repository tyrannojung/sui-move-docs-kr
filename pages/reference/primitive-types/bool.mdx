# Bool

`bool`은 불리언 `true`와 `false` 값을 위한 Move의 원시 타입입니다.

## 리터럴

`bool`의 리터럴은 `true` 또는 `false`입니다.

## 연산

### 논리 연산

`bool`은 세 가지 논리 연산을 지원합니다:

| 구문                      | 설명                         | 동등한 표현식                                                       |
| ------------------------- | ---------------------------- | ------------------------------------------------------------------- |
| `&&`                      | 단락 평가 논리 AND           | `p && q`는 `if (p) q else false`와 동등                            |
| <code>&vert;&vert;</code> | 단락 평가 논리 OR            | <code>p &vert;&vert; q</code>는 `if (p) true else q`와 동등        |
| `!`                       | 논리 부정                    | `!p`는 `if (p) false else true`와 동등                             |

### 제어 흐름

`bool` 값은 Move의 여러 제어 흐름 구조에서 사용됩니다:

- [`if (bool) { ... }`](./../control-flow/conditionals)
- [`while (bool) { .. }`](./../control-flow/loops)
- [`assert!(bool, u64)`](./../abort-and-assert)

## 소유권

언어에 내장된 다른 스칼라 값과 마찬가지로, 불리언 값은 암시적으로 복사 가능합니다.
즉, [`copy`](.././variables#move-and-copy)와 같은 명시적인 명령 없이도 복사될 수 있습니다.
